Librería Core

ScriptHook Rdr2 2.5

La biblioteca fundamental para la inyección de scripts personalizados dentro del juego. Permite ejecutar archivos y plugins nativos de extensión (.asi) de manera fluida y con compatibilidad completa para la versión 2.5.

  • Soporte completo para la ejecución de funciones nativas del motor ejecutor.
  • Carga automatizada de plugins ASI externos sin alterar binarios base.
  • Estructurado para prevenir conflictos de asignación de memoria (CTD).
  • Ideal para soportar entornos de pruebas personalizados en tiempo real.

Gestor de Recursos

Lennys Mod Loader

La herramienta avanzada definitiva (LML) que permite reemplazar recursos internos del juego sin necesidad de editar directamente los archivos contenedores (.rpf). Esencial para modificar de forma limpia texturas, modelos y densidades de fauna.

  • Reemplazo dinámico e inteligente de datos en tiempo de carga.
  • Carga simplificada y jerárquica mediante carpetas organizadas.
  • Soporte completo para modificaciones de densidad de entorno y calidad de objetos.
  • Totalmente compatible con gestores de modificaciones de comunidad.
